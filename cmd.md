以下命令将远程执行在同学电脑上！！！

关闭chrome浏览器(谷歌浏览器)【推荐执行这个命令】
```
taskkill -f -im chrome.exe
```
弹窗
```
msg [消息]
```
锁屏
```
lock
```
关机
```
shutdown -s -t [时间，单位为秒，至少为60]
```
重启
```
shutdown -r -t [时间，单位为秒，至少为60]
```
取消关机/重启:
```
shutdown -a
```
关闭edge浏览器
```
taskkill -f -im msedge.exe
```
